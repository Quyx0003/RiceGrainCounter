# Rice Grain Counter

## Description
This program counts the number of rice grains in a given image using image processing techniques. It takes an input image file and returns the count of rice grains present in the image.

## Prerequisites
- C++ compiler
- OpenCV library
- CMake (version 3.5 or higher)

## Installation
1. Clone the repository:

    git clone https://github.com/quyx0003/RiceGrainCounter.git
    
2. Navigate to the project directory:
    
    cd RiceGrainCounter
    
3. Create a build directory:
    
    mkdir build
    cd build
    
4. Generate Makefiles using CMake:
    
    cmake ..
    
5. Compile the program:
    
    make
    

## Usage

./CountRice <imageFile>
